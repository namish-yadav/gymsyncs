## [1.0.0] - Initial Release

### 🎉 Project Started

The first public version of GymSync.

This release establishes the foundation of the application using only HTML, CSS and Vanilla JavaScript.

### ✨ Added

#### Project
- Initial project structure
- Git repository
- GitHub repository
- Netlify deployment
- README documentation

#### User Experience
- First-time onboarding
- Name input
- Local Storage integration
- Welcome screen
- Responsive layout
- Mobile-first design

#### Dashboard
- Current date
- Current day
- Live clock
- Workout split
- Start Workout button

#### Workout System
- Push/Pull/Legs workout split
- Automatic workout selection based on current day
- Workout timer
- Exercise cards
- Weight input
- Notes input
- Exercise completion tracking

#### Design
- Dark mode interface
- Glassmorphism cards
- Rounded UI components
- Smooth animations
- Premium typography
- Modern responsive layout

#### Storage
- Offline functionality
- Local Storage persistence
- Saved user name
- Saved workout progress

During this version I learned:

### 🛠 Fixed
